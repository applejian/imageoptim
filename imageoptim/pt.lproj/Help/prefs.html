Preferências do ImageOptim

Não deverá ser necessário alterar qualquer destas opções (excepto o caminho para o PNGOUT caso o tenha instalado).

Preferências gerais

Preservar permissões, atributos e hardlinks para ficheiros

Em vez de apagar o ficheiro antigo e escrever um novo, irá apenas substituir o conteudo do ficheiro anterior. Isto assegura que todos os hardlinks, aliases, proprietários, permissoões e atributos adicionais do ficheiros são mantidos.

É ligeiramente mais rápido não preservar as permissões.

Criar cópia de segurança do ficheiro original antes de guardar

Irá criar uma cópia do ficheiro com ~ adicionado ao nome. Isto torna a optimização um pouco mais lenta e acumula cópias de segurança no seu disco rígido.

Número de tarefas simultâneas

Se tem um sistema multi-CPU ou multi-núcleo, executar mais do que uma tarefa simultaneamente irá acelerar um pouco a optimização.

Executar tarefas com prioridade baixa

Previne que todo o sistema fique lento durante o processo de optimização.

jpegoptim

JPEGOptim

Activado

Se assinalado, o ImageOptim irá tentar optimizar ficheiros JPEG.

Caminho para JPEGOptim

Caminho para o ficheiro executável jpegoptim. Normalmente /usr/bin/jpegoptim ou /sw/bin/jpegoptim.

Qualidade máxima

Se definido como 100%, a optimização será sem perdas.

Qualquer valor abaixo de 100% causará redução de qualidade nos ficheiros, se parecerem ter sido guardados com uma definição de qualidade máxima mais elevada.

Remover comentários dos ficheiros
Remover marcadores EXIF dos ficheiros

Remove metadados da imagem, como o nome do programa ou modelo de câmara que criou o ficheiro.

pngout

PNGOUT

Activado

Se assinalado, o ImageOptim irá tentar optimizar ficheiros PNG utilizando o PNGOUT.

O PNGOUT não é distribuído com o ImageOptim. É uma excelente ferramenta, mas o autor do PNGOUT proíbe a sua redistribuição, por isso terá de o descarregar manualmente.

Caminho para PNGOUT

Caminho para o ficheiro executável pngout (ou pngout-darwin).

Tipo de optimização

O PNGOUT é muito eficaz mas também muito lento. Utilize o nível mais baixo se for impaciente.

Interromper se demorar demasiado tempo a executar

O PNGOUT poderá necessitar de imenso tempo para optimizar imagens muito grandes. Esta opção irá interromper discretamente o PNGOUT após 1 minuto.

Remover todos os pedaços opcionais

Mantenha activo se estiver a optimizar para a Web. Desactive se tiver PNGs especiais com metadados adicionais.

pngcrush

PNGCrush

Activado

Se assinalado, o ImageOptim irá tentar optimizar ficheiros PNG utilizando o PNGCrush.

Caminho para PNGCrush

Caminho para o ficheiro executável pngcrush.

Tentar reparar ficheiros danificados

O PNGCrush não irá rejeitar ficheiros PNG que tenham somas de verificação (checksums) inválidas

Remover pedaços

O formato PNG guarda informação em pedaços e muitos deles são opcionais. Introduza os nomes de 4 letras dos pedaços que deseja remover ou pressione o botão γ para preencher a lista com os pedaços relacionados com valores de gama.

advpng

AdvPNG

Activado

Se assinalado, o ImageOptim irá tentar optimizar ficheiros PNG utilizando a ferramenta AdvPNG.

Caminho para AdvPNG

Caminho para o ficheiro executável advpng. Normalmente /usr/bin/advpng ou /sw/bin/advpng.

Nível de optimização
O AdvPNG é relativamente rápido, por isso pode utilizar o nível máximo.
optipng

OptiPNG

Activado

Se assinalado, o ImageOptim irá tentar optimizar ficheiros PNG utilizando a ferramenta OptiPNG.

Caminho para OptiPNG

Caminho para o ficheiro executável optipng. Normalmente /usr/bin/optipng ou /sw/bin/optipng.

Nível de optimização

O número de diferentes combinações de definições que serão testadas. Raramente fará sentido testá-las todas.

Tornar imagens entrelaçada (progressivas)

As imagens entrelaçadas têm melhor aparência quando estão parcialmenre carregadas, mas normalmente o entrelaçamento torna a compressão menos eficaz .

Esta caixa de verificação tem três estados: se assinalada, irá forçar as imagens a tornarem-se entrelaçadas. Se no estado intermédio, irá manter os ficheiros inalterados em termos de entrelaçamento. Se desassinalado, irá removar entrelaçamento dos ficheiros.

gifsicle

Gifsicle

Activado

Se assinalado, o ImageOptim irá tentar optimizar ficheiros GIF utilizando a ferramenta Gifsicle.

Caminho para Gifsicle

Caminho para o ficheiro executável gifsicle.

Ficheiros GIF serão alterados para entrelaçados ou não entrelaçados automaticamente, o que resultar num ficheiro menor. Comentários no ficheiro serão removidos.

Por favor tenha em conta que o Gifsicle funciona melhor com GIFs animados e que GIFs não-animados raramente podem ser optimizados.

Para melhor compressão, deve converter ficheiros GIF para PNGs (O ImageOptim não o faz automaticamente).
